# 2DRogue-lite
## 📄 [과제 기획서] 2D 무한 서바이벌 슈팅 게임

### 1. 개요

- **장르:** 2D Top-Down 슈팅 (로그라이트 서바이벌)
- **플랫폼:** PC (1920x1080 해상도 기준)
- **목표:** 사방에서 몰려오는 적들을 피해 최대한 오래 살아남고 높은 점수를 기록하는 것.
- **학습 목표:**
    - 2D 물리 이동 및 충돌 처리 (`Rigidbody2D`, `Collider2D`)
    - 프리팹(Prefab) 생성과 파괴 (`Instantiate`, `Destroy`)
    - 벡터 연산 기초 (적의 추적 알고리즘)
    - 게임 매니저를 통한 상태 관리 (시작, 플레이, 게임오버)

---

### 2. 핵심 게임 루프 (Core Loop)

1. **시작:** 타이틀 화면에서 'Game Start' 버튼 클릭.
2. **플레이:** 플레이어 이동 → 적 생성 및 추적 → 자동/수동 공격으로 적 처치 → 점수 획득.
3. **종료:** 플레이어 HP가 0이 되면 게임 오버.
4. **결과:** 최종 점수 및 최고 점수(Best Score) 표시 → 'Restart' 버튼으로 재시작.

---

### 3. 세부 기능 요구사항

### A. 플레이어

- **이동:** WASD 또는 방향키로 상하좌우 이동 (물리 기반 이동 권장 `MovePosition` 또는 `Velocity`).
- **제한:** 화면 밖으로 나가지 못하도록 경계 처리 (Mathf.Clamp 또는 벽 Collider 사용).
- **공격 (택 1)**
    - *옵션 1 (쉬움):* 가장 가까운 적을 향해 일정 간격으로 총알 자동 발사.
    - *옵션 2 (보통):* 마우스 클릭 시 마우스 커서 방향으로 총알 발사.

### B. 적

- **생성:** 화면 밖 랜덤한 위치에서 일정 간격(예: 1초)마다 생성.
- **이동:** 생성되자마자 플레이어의 현재 위치를 향해 직선으로 계속 이동 (유도탄 아님, 플레이어 방향 벡터 구하기).
- **충돌**
    - 플레이어와 닿으면 → 플레이어 HP 감소 & 적 소멸.
    - 총알과 닿으면 → 적 소멸 & 점수 증가.

### C. 시스템 & UI

- **GameManager:** 게임의 상태 (Ready, Play, GameOver)를 관리하는 싱글톤 혹은 핵심 클래스 구현.
- **UI 표시:**
    - **HUD:** 현재 점수, 플레이어 HP 바(Bar) 또는 하트 아이콘.
    - **Game Over 팝업:** "Game Over" 텍스트, 최종 점수, 최고 점수(PlayerPrefs 저장), 재시작 버튼.

---

### 4. 레벨 디자인 & 밸런스

- **시간 경과 난이도:** 게임 시작 후 30초가 지날 때마다 적 생성 주기가 0.1초씩 빨라지게 구현 (최소 0.2초까지).
- **플레이어 스펙:** HP 3칸, 이동 속도 5.
- **적 스펙:** 한 방에 사망, 이동 속도 3 (플레이어보다 느리게).
